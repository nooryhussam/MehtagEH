# Mehtag Eh (محتاج إيه)

Mehtag Eh is a smart humanitarian aid platform that connects people in need with donors through a transparent and organized digital system. The application aims to simplify the aid process by helping donors find verified cases and ensuring that support reaches those who need it most.

## Overview

Traditional aid distribution often relies on manual processes, such as reviewing social media posts, verifying documents, and matching donors with beneficiaries. These methods can be time-consuming and difficult to manage.

Mehtag Eh provides a structured platform that improves efficiency, transparency, and accessibility by combining mobile technology with AI-powered features.

## Features

### ID Verification
Users can upload their national ID cards, and OCR technology automatically extracts and validates the information to reduce fraud and duplicate accounts.

### Smart Priority Scoring
Aid requests are evaluated and ranked based on factors such as request type, family size, and supporting documents, helping urgent cases receive attention faster.

### Intelligent Donor Matching
The platform recommends cases to donors based on their preferences, location, and donation interests.

### Real-Time Tracking
Both donors and beneficiaries can track the progress of aid requests from submission to delivery.

### Secure User Management
Authentication and profile management ensure that users can safely access and manage their accounts.

## User Flow

### Requester
1. Create an account
2. Upload ID for verification
3. Submit an aid request
4. Upload supporting documents
5. Track request status

### Donor
1. Create an account
2. Set donation preferences
3. Browse recommended cases
4. Support a case
5. Receive status updates

- Improve transparency in aid distribution.
